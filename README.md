# AI-Desktop-Automation-Bot

This is a Python automation / bot project built using:
- Python
- PyAutoGUI
- OpenAI API

## Files
- `bot.py` – main bot logic
- `cursor.py` – mouse cursor tracking using PyAutoGUI
- `openai_client.py` – OpenAI API client setup

## How to Run
```bash
pip install pyautogui openai
python cursor.py
